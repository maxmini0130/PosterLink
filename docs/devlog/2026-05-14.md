### 4-4. 업로드 서버 사이드 이관 (Network request failed 근본 수정)

**파일**
- `apps/web/app/api/upload/poster/route.ts` (신규)
- `apps/web/app/operator/posters/new/page.tsx`
- `apps/web/next.config.mjs`

**원인**
- 모바일 WebView에서 `Blob`을 직접 Supabase Storage에 `PUT` 요청 시 네트워크 레이어 미지원으로 `Network request failed` 발생

**해결**
- 브라우저: Blob → base64 변환 후 `/api/upload/poster`로 JSON POST (동일 도메인)
- 서버: base64 디코딩 → `service_role`로 Supabase Storage에 업로드 → `publicUrl` 반환
- `next.config.mjs`에 `serverActions.bodySizeLimit: "10mb"` 추가

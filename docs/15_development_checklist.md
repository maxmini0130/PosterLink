# 포스터링크 개발 체크리스트

## 1. 목표

이 문서는 바이브코딩으로 빠르게 구현을 시작할 때,
누락 없이 기술/기획/운영 기본 요소를 점검하기 위한 체크리스트다.

## 2. 프로젝트 초기 세팅

- [ ] 서비스명/영문명/슬로건 확정
- [ ] 메인 컬러/기본 브랜딩 방향 확정
- [ ] 모노레포 기본 구조 생성
- [ ] Next.js 웹 앱 생성
- [ ] Expo 모바일 앱 생성
- [ ] Supabase 프로젝트 생성
- [ ] 환경변수 구조 정리
- [ ] 공통 TypeScript 설정 정리
- [ ] 린트/포맷터 적용

- [ ] Supabase Auth 설정

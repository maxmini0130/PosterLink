# 포스터링크 브랜드 가이드 초안

## 1. 기본 정보

- 서비스명: **포스터링크**
- 영문명: **PosterLink**
- 프로젝트명: **공공포스터 링크 플랫폼**
- 메인 슬로건: **길에서 본 공공 포스터, 다시 찾다**

## 2. 서비스 정의

포스터링크는 길에서 스쳐 지나간 공공기관 홍보 포스터를 다시 찾고, 이해하고, 공식 신청 페이지까지 연결해 주는 서비스다.

단순 포스터 모음 앱이 아니라 아래 문제를 해결하는 데 초점을 둔다.

- 길에서 포스터를 봤지만 나중에 다시 찾기 어렵다.
- 구청/시청/복지관 홈페이지에 정보가 흩어져 있어 탐색 비용이 크다.
- 사용자는 본인 지역, 연령, 관심사에 맞는 공고를 우선적으로 보고 싶어 한다.

## 3. 브랜드 핵심 키워드

- 공공
- 신뢰
- 재탐색
- 연결
- 지역
- 맞춤추천
- 생활밀착

## 4. 브랜드 톤 앤 매너

### 기본 방향
- 공공정보 서비스답게 **신뢰감**이 있어야 한다.
- 시민 서비스답게 **친근함**이 있어야 한다.
- 모바일 서비스답게 **직관적**이어야 한다.
- 지나치게 행정 시스템처럼 딱딱하거나, 반대로 너무 가벼운 커뮤니티 앱처럼 보여서는 안 된다.

### 문장 스타일
- 짧고 명확하게 쓴다.
- 행정 용어는 최소화하고 생활형 표현을 우선한다.
- 단, 공식 정보 안내 문구에서는 정확성을 우선한다.

예시:
- 길에서 본 공공 포스터, 다시 찾다
- 내 지역에 맞는 공공 공고를 한눈에
- 포스터를 보고 끝내지 않도록

## 5. 로고 방향

### 추천 컨셉
**Poster Frame + Link**

포스터를 상징하는 사각형 프레임과, 온라인 연결을 상징하는 링크 요소를 결합한다.

### 의미
- 오프라인 포스터
- 온라인 정보 연결
- 끊긴 탐색 경험의 복원

### 디자인 방향
- 단순한 플랫 스타일
- 복잡한 장식 최소화
- 작은 앱 아이콘에서도 인식 가능한 구조
- 공공기관 제안서와 서비스 UI 모두에 자연스럽게 어울리는 형태

## 6. 앱 아이콘 방향

### 아이콘 요소
- 둥근 사각형 포스터 프레임
- 링크 모양 연결 포인트
- 텍스트는 넣지 않거나 최소화

### 주의사항
- 지나치게 입체적인 스타일 지양
- 작은 크기에서 식별 안 되는 디테일 지양
- 정부 공식 앱처럼 오인할 수 있는 상징은 피함

## 7. 컬러 가이드

### 메인 팔레트 추천
- Primary: **딥 블루 계열**
- Secondary: **소프트 민트 계열**
- Background: **라이트 그레이 화이트**
- Text Primary: **다크 슬레이트**
- Alert/Deadline: **코랄 레드 계열**

### 컬러 역할
- Primary: 주요 버튼, 헤더, 활성 상태
- Secondary: 선택 상태 강조, 보조 배지, 필터 강조
- Background: 앱 기본 배경
- Alert: 마감 임박, 신고, 주의 상태

### 사용 원칙
- 화면 전체를 블루로 덮지 않는다.
- 공고 본문은 충분한 명도 대비를 확보한다.
- 마감일, 경고, 비공개 상태는 색상만이 아니라 텍스트/아이콘으로도 구분한다.

## 8. 타이포그래피 방향

- 제목: 단정하고 명확한 고딕 계열
- 본문: 작은 화면에서도 잘 읽히는 산세리프
- 숫자/날짜: 마감일과 기간이 또렷하게 보이는 UI 폰트 톤 유지

핵심은 멋보다 **가독성 우선**이다.

## 9. 기본 소개 문구

### 소개문 1
포스터링크는 길에서 본 공공 포스터를 다시 찾고, 이해하고, 신청까지 연결하는 서비스입니다.

### 소개문 2
지역, 연령, 관심사에 맞는 공공 공고를 더 쉽게 발견할 수 있도록 돕습니다.

### 소개문 3
흩어진 공공 안내를 한곳에 모아, 시민이 놓치지 않도록 연결합니다.

### 기관/운영자용 설명
포스터링크는 공공기관 홍보 포스터를 디지털 링크와 연결해, 도달 이후의 탐색과 신청 전환까지 지원하는 플랫폼입니다.

## 10. 브랜드 금지사항

- 정부/국가/행정 등 오인 가능성이 높은 공식기관형 네이밍 남용 금지
- 지나치게 캐주얼한 밈 스타일 표현 지양
- 사용자 의견과 공식 정보를 시각적으로 동일하게 보이게 만드는 디자인 금지
- 과한 네온, 과한 그라디언트, 가독성을 해치는 장식 지양

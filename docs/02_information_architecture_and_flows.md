# 02. 정보구조(IA) 및 주요 플로우

## 1. 사용자 앱 IA

### 하단 탭 구조
1. 홈
2. 탐색
3. 찜
4. 알림
5. 마이

## 2. 사용자 앱 상세 메뉴

### 홈
- 맞춤 추천 영역
- 마감 임박 공고
- 최신 등록 공고
- 카테고리 퀵 진입
- 지역 변경 CTA

### 탐색
- 검색창
- 카테고리 필터
- 지역 필터
- 대상 필터
- 정렬(추천순 / 마감임박순 / 최신순 / 인기순)
- 포스터 리스트

### 찜
- 진행중
- 마감임박
- 마감됨
- 유사 공고 추천

### 알림
- 찜한 공고 마감 임박 알림
- 관심 카테고리 신규 등록
- 운영자 공지
- 댓글 답변/대댓글 알림

### 마이
- 내 정보
- 관심 조건 수정
- 찜 관리
- 댓글 관리
- 신고 내역
- 알림 설정
- 로그아웃

## 3. 포스터 상세 IA

- 포스터 이미지
- 제목
- 기관명
- 등록일
- 대상 조건
- 신청 기간
- 지역
- 카테고리
- 요약 정보(3줄 요약)
- 공식 홈페이지 링크
- 신청 페이지 링크(있을 경우)
- 참고 링크(블로그/보도자료 등)
- 찜하기
- 공유
- 질문/후기(댓글)
- 유사 포스터 추천

## 4. 운영자 도구 IA

### 운영자 홈
- 오늘 등록 건수
- 임시 저장
- 검수 대기
- 반려 건수
- 최근 등록 목록

### 등록 프로세스
1. 포스터 촬영/업로드
2. 이미지 보정 확인
3. OCR 검수
4. 메타데이터 입력
5. 링크 연결
6. 미리보기
7. 게시 승인 요청 / 저장

## 5. 관리자 CMS IA

### 대시보드
- 총 포스터 수
- 진행중/마감/비공개 수
- 댓글 수
- 신고 대기 수
- 오늘 등록/수정/삭제 수

### 포스터 관리
- 목록
- 검색
- 상태 변경
- 수정/삭제
- 만료 처리
- 중복 확인

### 댓글/신고 관리
- 전체 댓글 목록
- 신고 접수 목록
- 숨김/삭제 처리
- 사용자 제재

### 기준정보 관리
- 카테고리
- 지역 코드
- 대상 조건 코드
- 금칙어
- 공지사항

## 6. 사용자 주요 플로우

### 플로우 A: 첫 가입
1. 회원가입/소셜로그인
2. 지역 선택
3. 연령대 선택
4. 관심 카테고리 선택
5. 선택 조건 저장
6. 홈 진입

### 플로우 B: 포스터 탐색
1. 홈 또는 탐색 진입
2. 필터/검색 수행
3. 목록 확인
4. 포스터 상세 진입
5. 찜/공식링크 이동/댓글 확인

### 플로우 C: 찜 중심 재방문
1. 찜 알림 수신
2. 앱 재진입
3. 찜 목록 확인
4. 마감일 또는 공식 링크 재확인

## 7. 운영자 주요 플로우

### 플로우 D: 등록
1. 이미지 업로드
2. 보정본 승인
3. OCR 결과 검수
4. 포스터 정보 입력
5. 링크 입력
6. 임시저장 또는 게시 요청

### 플로우 E: 수정
1. 기존 포스터 검색
2. 상세 편집
3. 수정 사유 입력
4. 저장
5. 변경 이력 기록

## 8. 관리자 주요 플로우

### 플로우 F: 신고 처리
1. 신고 리스트 확인
2. 원 댓글/대댓글 확인
3. 신고 사유 확인
4. 숨김/삭제/유지 결정
5. 필요 시 사용자 제재
6. 처리 이력 기록

### 플로우 G: 포스터 만료 처리
1. 마감일 지난 포스터 자동 후보 확인
2. 관리자 검수
3. 마감 상태 전환
4. 사용자 노출 위치 변경(기본 피드 제외)

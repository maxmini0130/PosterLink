# 07. 댓글 및 운영정책

## 1. 댓글 기능 목적

댓글은 일반 커뮤니티 활성화가 목적이 아니다.  
이 서비스에서 댓글은 다음 역할을 수행한다.

1. 질문
2. 후기 공유
3. 정보 보완
4. 정정 요청

즉, **공식 정보 보조 채널**로 운영한다.

## 2. 기본 원칙

1. 공식 정보와 사용자 의견을 명확히 구분한다.
2. 댓글이 공고 원문을 대체하면 안 된다.
3. 자격 조건 판단은 공식 링크 기준으로 안내한다.
4. 허위정보, 광고, 분쟁 유발 댓글은 신속히 조치한다.

## 3. 허용되는 댓글 예시

- “재직자도 신청 가능한지 궁금합니다.”
- “직접 신청해보니 서류 제출 단계가 추가로 있었습니다.”
- “공식 링크가 바뀐 것 같습니다.”
- “저는 신청 완료했습니다. 일정 참고하세요.”

## 4. 제한/금지 댓글 예시

- 욕설, 비방, 혐오 표현
- 정치적 분쟁 유도
- 기관/개인 실명 비방
- 광고/홍보/도배
- 개인정보 노출
- 확인되지 않은 허위 자격 판단
- 외부 유료 서비스 유도

## 5. 신고 사유 체계

- `abuse`: 욕설/비방
- `misinformation`: 허위정보
- `spam`: 광고/도배
- `political`: 정치/분쟁 유도
- `privacy`: 개인정보 노출
- `hate`: 혐오표현
- `other`: 기타

## 6. 댓글 상태 정의

- `normal`: 정상 노출
- `hidden`: 관리자 숨김
- `deleted`: 삭제
- `blocked`: 차단된 상태

## 7. 관리자 조치 유형

- 댓글 숨김
- 댓글 삭제
- 경고
- 일정기간 작성 제한
- 영구 정지(중대한 위반 시)
- 신고 기각

## 8. 운영 가이드

### 운영자/관리자 답변 문구 원칙
- 자격 판단은 단정적으로 말하지 않는다.
- “공식 공고 기준 확인 필요” 문구를 유지한다.
- 오해 소지가 큰 경우 링크를 함께 제시한다.

### 권장 문구 예시
“상세 자격과 절차는 공식 공고문 기준으로 확인이 필요합니다. 아래 공식 링크를 참고해 주세요.”

## 9. 사용자 화면 정책 문구

포스터 상세 및 댓글 상단에 아래 문구를 노출한다.

> 정확한 신청 자격 및 절차는 공식 공고를 반드시 확인하세요.

## 10. 신고 처리 SLA(권장)

- 일반 신고: 24시간 이내 검토
- 허위정보/분쟁성 신고: 우선 검토
- 개인정보 노출: 즉시 숨김 우선

# 03. 사용자 앱 화면 정의

## 화면 목록

1. 스플래시 / 온보딩
2. 로그인 / 회원가입
3. 관심 정보 설정
4. 홈
5. 탐색
6. 포스터 상세
7. 찜 목록
8. 알림
9. 마이페이지
10. 내 댓글 관리

---

## 1. 관심 정보 설정 화면

### 목적
개인화 추천을 위한 최소 조건 수집

### 입력 항목
- 지역(필수)
- 연령대(필수)
- 관심 카테고리(필수, 복수 선택 가능)
- 성별(선택)
- 추가 조건(선택)
  - 1인가구
  - 구직중
  - 소상공인
  - 학부모/자녀 있음
  - 학생
  - 시니어 관련 대상 등

### 액션
- 저장 후 추천 시작

### 유의사항
초기 가입 이탈을 줄이기 위해 필수값은 3개만 유지

---

## 2. 홈 화면

### 목적
사용자 조건에 맞는 포스터를 우선 노출

### 상단 영역
- 사용자 지역/관심 요약
- 검색 진입
- 알림 아이콘

### 본문 섹션
- 맞춤 추천
- 마감 임박
- 최신 등록
- 전체 대상 인기 공고
- 카테고리 바로가기

### 카드 정보
- 썸네일
- 제목
- 기관명
- 마감일
- 태그(지역/연령/카테고리)
- 찜 수/댓글 수(선택)

### 정렬 옵션
- 추천순
- 마감임박순
- 최신순
- 인기순(2차)

---

## 3. 탐색 화면

### 목적
검색/필터 기반 탐색

### 필수 요소
- 검색창
- 카테고리 필터
- 지역 필터
- 대상 조건 필터
- 정렬

### 검색 대상
- 포스터 제목
- 기관명
- 지역명
- 태그

### 필터 예시
- 지역: 전국 / 시도 / 시군구
- 연령: 전체 / 청소년 / 청년 / 중장년 / 시니어
- 성별: 전체 / 여성 / 남성 / 무관
- 카테고리: 복수 선택
- 마감 상태: 진행중 / 마감임박 / 전체

---

## 4. 포스터 상세 화면

### 목적
공고 내용을 빠르게 이해하고 행동 유도

### 섹션 구성
1. 포스터 이미지
2. 제목 / 기관명 / 등록일
3. 핵심 요약
4. 대상/기간/지역/카테고리
5. 공식 링크
6. 참고 링크
7. 찜 / 공유
8. 질문/후기(댓글)
9. 유사 포스터

### 핵심 정보 블록
- 대상
- 신청 기간
- 신청 방법
- 문의처
- 지역 범위
- 혜택 내용(있을 경우)

### CTA
- 공식 홈페이지 보기
- 신청 페이지 바로가기
- 찜하기
- 알림 받기

### 정책 문구
“정확한 신청 자격과 절차는 공식 공고를 반드시 확인하세요.”

---

## 5. 찜 목록 화면

### 목적
재방문 유도 및 저장 공고 관리

### 탭/구분
- 진행중
- 마감임박
- 마감됨

### 액션
- 찜 해제
- 상세 보기
- 마감 알림 설정/해제

### 보조 기능
- 찜한 공고와 유사한 새 공고 추천

---

## 6. 알림 화면

### 알림 종류
- 찜한 포스터 마감 임박
- 관심 카테고리 신규 등록
- 댓글 답변
- 대댓글
- 운영 공지

### 상태
- 전체 / 안읽음

---

## 7. 마이페이지

### 항목
- 내 프로필
- 관심 조건 수정
- 알림 설정
- 찜한 포스터
- 내 댓글
- 신고 내역
- 이용약관 / 운영정책
- 로그아웃

---

## 8. 댓글 UI 원칙

### 댓글 영역 이름
- 댓글 또는 질문/후기

### 댓글 유형(권장 2차)
- 질문
- 후기
- 정보공유
- 정정요청

### 댓글 표시 요소
- 작성자 닉네임
- 작성 시각
- 본문
- 신고 버튼
- 답글 버튼
- 운영자/공식 답변 배지

### 중요 원칙
공식 정보 블록과 댓글 블록은 UI 상에서 명확히 분리한다.

# 04. 운영자/관리자 화면 정의

## A. 운영자 등록도구

### 1. 운영자 홈
#### 목적
당일 작업 현황 확인 및 빠른 등록 시작

#### 표시 항목
- 오늘 등록 완료 건수
- 임시 저장 건수
- 검수 대기 건수
- 반려 건수
- 최근 수정 건수

#### CTA
- 새 포스터 등록
- 임시 저장 이어서 작업
- 반려 사유 확인

---

### 2. 촬영/업로드 화면
#### 목적
포스터 원본 이미지 수집

#### 기능
- 카메라 촬영
- 갤러리 업로드
- 다중 업로드(2차)
- 촬영 가이드(선택)
- 위치 자동 기록(선택)

#### 저장 데이터
- 원본 이미지
- 촬영 시각
- 촬영자
- 위치 정보(옵션)

---

### 3. 이미지 보정 확인 화면
#### 목적
보정본을 확인하고 다음 단계로 이동

#### 기능
- 원본/보정본 토글
- 자동 크롭 결과 확인
- 회전/자르기 수동 보정
- 밝기/대비/선명도 보정(기본값 제공)
- 재업로드

#### 상태
- 임시 저장 가능

---

### 4. OCR 검수 화면
#### 목적
이미지에서 추출한 텍스트 확인 및 수정

#### 자동 추출 대상
- 제목
- 기관명
- 기간
- 문의처
- URL/QR

#### 입력 필드
- 제목
- 기관명
- 요약
- 문의처
- 기간 텍스트 원문
- 메모

---

### 5. 메타데이터 입력 화면
#### 목적
포스터를 탐색 가능한 구조화 데이터로 변환

#### 입력 항목
- 카테고리
- 지역(전국 / 시도 / 시군구)
- 대상 연령대
- 성별 조건
- 추가 자격 조건
- 노출 시작일
- 노출 종료일
- 태그
- 검색 키워드

#### 상태값
- 임시저장
- 검수요청
- 게시중
- 반려
- 마감

---

### 6. 링크 연결 화면
#### 목적
공식 정보와 참고 정보를 분리 등록

#### 공식 링크
- 공식 공고 URL
- 신청 URL
- 기관 홈페이지 URL

#### 참고 링크
- 보도자료
- 블로그
- 유튜브
- 기타 참고 URL

#### 링크 정책
공식 링크 최소 1개 이상 필수

---

### 7. 미리보기/게시 요청 화면
#### 목적
최종 사용자 화면 미리보기 후 게시

#### 확인 항목
- 썸네일
- 제목
- 태그
- 요약
- CTA 버튼
- 댓글 허용 여부(선택)
- 공개 범위

#### 액션
- 임시저장
- 게시 요청
- 바로 게시(권한자만)

---

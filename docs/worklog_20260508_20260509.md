# PosterLink 작업 로그 - 2026-05-08 ~ 2026-05-09

## 요약

2026년 5월 8일부터 5월 9일까지는 PosterLink의 운영 자동화, 크롤러 품질 개선, 관리자 검수 기능, 포스터 이미지 판별, 사용자 목록 필터링을 집중적으로 진행했다.

핵심 흐름은 다음과 같다.

- 사용자 포스터 등록 요청과 포인트 보상 구조 추가
- 모바일 FAB 동작과 맞춤 설정 온보딩 개선
- 마포구 및 공공기관 크롤러 구축
- GitHub Actions 일일 크롤러 배치 추가
- Supabase 업로드, 중복 처리, 이미지 필터링, 카테고리 자동 분류 개선
- 관리자 검수 화면의 대량 승인/반려, 상세 미리보기, 반려 삭제 기능 추가
- 포스터 이미지가 아닌 항목을 줄이기 위한 규칙 기반/AI 기반 필터 도입
- 메인/목록 화면의 카운트, 개인화 필터, 정렬 기능 개선

### 5. 크롤러 중복 처리 개선

주요 변경:

- `source_key` 정규화
- 페이지 번호, 정렬, 임시 query parameter 제거

- 기존 중복 포스터에도 카테고리 연결 보강

현재 동작:

- 같은 원본 게시글은 다음 크롤링에서 중복으로 새로 생성되지 않음
- 이미 있는 포스터는 상태를 유지
- 새로 들어온 포스터만 `review` 상태로 들어감

관련 커밋:

- 마감 제외/마감 포함 상태에 따라 숫자가 달라지도록 수정
- 사용자 조건에 의해 노출되는 목록과 카운트가 일치하도록 정리

관련 커밋:

사용자가 전체 포스터를 볼 수 있으면서도, 필요할 때 본인 조건에 맞는 포스터만 볼 수 있도록 `/posters` 목록을 개선했다.

주요 변경:

- 로그인 사용자의 기본 지역 또는 관심 카테고리와 맞는 포스터만 필터 가능
- 정렬 추가
  - 최신
  - 마감임박
  - 인기
  - 조회
  - 찜
  - 클릭
- 마감 포함 상태에서도 마감임박 정렬 시 이미 마감된 포스터가 위로 올라오지 않도록 처리
- 포스터의 첫 번째 카테고리/지역뿐 아니라 전체 연결 카테고리/지역을 기준으로 필터링

# 00. 서비스 개요

## 서비스명(가칭)
공공 포스터 앱 / 공공 공고 포스터 모음 서비스

## 한 줄 정의
길에서 본 공공기관 포스터를 **다시 찾고, 이해하고, 저장하고, 놓치지 않게 만드는 개인 맞춤형 공공정보 서비스**

## 문제 정의

공공기관은 포스터를 제작해 오프라인 게시판, 지하철역, 구청, 주민센터, 복지관 등에 붙인다.  
시민은 지나가며 내용을 보지만 나중에 다시 찾기 어렵다.

문제는 크게 4가지다.

1. 포스터를 그 자리에서 자세히 읽기 어렵다.
2. 나중에 기억이 나도 어느 기관 홈페이지에 올라왔는지 찾기 어렵다.
3. 홈페이지에 올라와 있어도 메뉴 구조가 복잡해 재탐색 비용이 크다.
4. 나와 관련된 공고만 모아보기가 어렵다.

## 해결 방식

이 서비스는 다음 과정을 제공한다.

1. 맥스미니 운영자가 현장에서 포스터를 촬영한다.
2. 포스터 이미지를 보정하고, 텍스트를 구조화한다.
3. 공식 공고 링크와 참고 링크를 연결한다.
4. 사용자 프로필(지역, 연령대, 관심 카테고리, 조건)에 맞는 포스터를 우선 노출한다.
5. 찜, 마감 알림, 질문/후기, 댓글 신고 기능을 제공한다.

## 핵심 가치

### 1) 다시 찾을 수 있음
오프라인에서 본 정보를 온라인에서 쉽게 재탐색 가능

### 2) 이해하기 쉬움
긴 공고문 대신 대상, 신청 기간, 방법, 혜택을 요약 제공

### 3) 개인화
내 지역, 내 연령대, 내 관심사 기준으로 우선 노출

### 4) 운영 가능성
초기에는 맥스미니가 직접 수집/검수하여 품질 보장 가능

## 주요 사용자

### A. 일반 사용자
- 지역 기반 지원사업/교육/행사/복지 정보를 찾는 시민
- 청년/시니어/부모/소상공인 등 특정 조건 대상자
- 찜/알림을 통해 마감 전에 다시 보고 싶은 사용자

### B. 운영자
- 포스터 현장 수집 담당
- 포스터 이미지 보정, OCR 검수, 링크 연결, 게시 승인 담당

### C. 관리자
- 서비스 전체 포스터 품질 관리
- 댓글/신고/운영 정책 처리
- 카테고리/지역/조건값 관리

## 핵심 기능 범주

### 사용자 기능
- 회원가입 / 로그인
- 지역, 연령대, 관심 카테고리 설정
- 추천 피드
- 검색 / 필터 / 정렬
- 포스터 상세 조회
- 찜하기
- 댓글 / 대댓글
- 신고
- 마감 알림

### 운영 기능
- 포스터 촬영 업로드
- 자동/반자동 보정
- OCR 추출
- 공식 링크 입력
- 카테고리/지역/대상 조건 지정
- 게시 승인

### 관리자 기능
- 포스터 수정/비공개/삭제
- 댓글/대댓글 숨김/삭제
- 신고 처리
- 금칙어 관리
- 카테고리 관리
- 사용자 제재

## 권장 MVP 정의

초기 MVP는 아래에 집중한다.

- 맥스미니 직접 수집형 운영 모델
- 사용자 맞춤 피드
- 찜하기
- 기본 댓글
- 관리자 신고 처리
- 운영자 등록도구

## 성공 기준(초기)

### 운영 KPI
- 1건 등록 평균 소요시간
- 운영자 1인당 일일 등록 건수
- 포스터 당 공식 링크 연결률
- 기간/지역/카테고리 구조화 완료율

### 사용자 KPI
- 홈 진입 후 상세 진입률
- 찜 전환율
- 찜 후 재방문율
- 마감 임박 공고 클릭률
- 댓글/질문 참여율
